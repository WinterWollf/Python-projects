## How to Run

1. Clone the repository:
    ```sh
    git clone https://github.com/WinterWollf/Python.git
    cd Python-projects
    ```

2. Navigate to the project directory you want to run:
    ```sh
    cd "Project Name"
    ```

3. Run the main Python file:
    ```sh
    python main.py
    ```
